# MovieDatabaseSystem
## A simple Java console application simulating a movie database system

### Functions:
- Read data from local file
- Add movies
- Delete movies
- Modify movies info
- Search movies
- Save data to local file 

### File Structure
- models
  - Actor.java
  - Movie.java
  - MovieList.java
- ui
  - MovieDatabase.java
- helperLibrary
  - ValidationProcessor.java
  - StandardMessage.java
- Initial.java
